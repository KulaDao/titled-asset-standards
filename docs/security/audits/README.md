## Adding a report

1. Save the **final published PDF** here (not drafts or redlines).
2. Use a URL-friendly filename: `{auditor}-titled-asset-standards-v{major.minor}.pdf`
   - Example: `verichains-titled-asset-standards-v1.1.pdf` after remediation re-review.
3. Add a row to the table above with the pinned commit SHA and issue date.
4. Update [../README.md](../README.md) with the canonical auditor/scope/commit fields.

Do **not** commit internal remediation spreadsheets, finding-by-finding writeups, or links to private document stores.
